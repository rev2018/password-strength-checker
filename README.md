# Password Strength Analyzer

A web application that analyzes the strength and security of user passwords using entropy calculation, crack-time estimation, and real-time validation.

This tool helps users understand how secure their passwords are and encourages better password practices through immediate visual feedback.

## Features

- Real-time password strength analysis
- Password entropy calculation
- Estimated password crack time
- Detection of commonly used passwords
- Password generator for creating secure passwords
- Copy password functionality with notification
- Show / hide password toggle
- Dark mode support
- Responsive and minimal UI design

## How It Works

The application evaluates a password based on several factors such as:

- Length of the password  
- Presence of uppercase letters  
- Presence of lowercase letters  
- Numeric characters  
- Special characters  

Using these factors, the tool calculates password entropy and estimates how long it might take for the password to be cracked using brute-force techniques.

## Tech Stack

- HTML  
- CSS  
- JavaScript  

## Purpose and Impact

Weak passwords are one of the most common causes of security breaches.  
This project demonstrates how front-end web technologies can be used to help users create stronger and more secure passwords through interactive feedback and analysis.

It also serves as a practical implementation of password security concepts such as entropy, password complexity, and strength evaluation.
